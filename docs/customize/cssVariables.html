CSS Variables

Maeven will set most theme options als css variables so consistent styling is easy to extend in your own custom classes. Below a list of all Maeven CSS variables.

Base#

NameDescription
--mvn-baseBase size for all theme. This is the value 1rem will be. defaults to 10px

Animations#

NameDescription
--mvn-transition-durationDefault duration for transitions
--mvn-transition-timing-functionDefault timing function for transitions
