# Getting Started

## Install

**Maeven** can by installed by yarn or npm. Make sure you also add `react` and `react-dom` which are peer dependencies.

### Yarn

```bash
yarn add maeven react react-dom
```

### npm

```bash
npm install maeven react react-dom
```

### Import

Import required React components from Maeven.

```jsx
import { Button } from 'maeven';

<Button type="primary" size="large">
  Hello World!
</Button>;
```
