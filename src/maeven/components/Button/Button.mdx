```js live

```

### buttonType

Buttons can by styled as different type of buttons.

```js live
<Block padding>
  <P>
    <Button buttonType="primary">Primary</Button> <Button>Default</Button>{' '}
    <Button buttonType="link">Link</Button>{' '}
    <Button buttonType="success">Success</Button>{' '}
    <Button buttonType="warning">Warning</Button>{' '}
    <Button buttonType="danger">Danger</Button>
  </P>
  <P>
    <Button disabled buttonType="primary">
      Disabled Primary
    </Button>{' '}
    <Button disabled>Disabled Default</Button>{' '}

      Disabled Danger
    </Button>
  </P>
</Block>
```

### Loading

Buttons can have a loading state.

```js live withRender
// import { refreshCw } from 'icon-packs/feather';

const Example = () => {
  const [loading, setLoading] = useState(false);

  useEffect(() => {
    if (loading) {
      const to = setTimeout(() => {
        setLoading(false);
      }, 3000);
    }
  }, [loading]);

  return (
    <Button
      buttonType="primary"
      icon={refreshCw}
      loading={loading}
      onClick={() => setLoading(true)}
    >
      Click to toggle
    </Button>
  );
};

render(<Example />);
```

### Sizes

Buttons can have different sizes.

```js live
<>
  <Button icon={userCircleSolid} buttonType="primary" size="sm">
    Small Button
  </Button>{' '}
  <Button icon={userCircleSolid} buttonType="primary" size="md">
    Medium Button
  </Button>{' '}
  <Button icon={userCircleSolid} buttonType="primary" size="lg">
    Large Button
  </Button>
</>
```

### Fluid

A fluid button takes up the full width.

```js live
<Button buttonType="primary" fluid size="lg">
  Fluid Button
</Button>
```

### Active

A button can be styled to indacte it is active.

```js live
<>
  <Button buttonType="primary">Normal</Button>{' '}
  <Button buttonType="primary" active>
    Active
  </Button>
</>
```
