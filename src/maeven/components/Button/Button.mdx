### Basic

A basic usage of Button.

```js live
<Button onClick={() => alert('Button Clicked')}>Click Me!</Button>
```

### buttonType

```js live

  <P>
    <Button buttonType="primary">Primary</Button>{' '}
    <Button buttonType="secondary">Secondary</Button>{' '}
    <Button buttonType="link">Link</Button>{' '}
    <Button buttonType="info">Info</Button>{' '}
    <Button buttonType="success">Success</Button>{' '}
    <Button buttonType="warning">Warning</Button>{' '}
    <Button buttonType="danger">Danger</Button>
  </P>
  <P>
    <Button disabled buttonType="primary">
      Disabled Primary
```

A button can contain icons.

```js live

<>
  <Button buttonType="primary" icon={refreshCw}>
    Refresh
  </Button>{' '}
  <Button icon={userCircleSolid} iconRight={caretDownSolid}>

  </Button>{' '}
  <Button buttonType="warning" iconRight={arrowRightSolid}>
    Next
  </Button>{' '}
  <Button icon={fileRegular} iconRight={timesSolid}>
    Upload...
  </Button>{' '}

  <Button icon={arrowLeftSolid} iconRight={arrowRightSolid} />

```

### Sizes

Buttons can have different sizes.

```js live
<>
  <Button buttonType="primary" size="sm">
    Small Button
  </Button>{' '}
  <Button buttonType="primary" size="md">
    Medium Button
  </Button>{' '}
  <Button buttonType="primary" size="lg">
    Large Button
  </Button>
</>
```

### Fluid

A fluid button takes up the full width.

```js live
<Button buttonType="primary" fluid>
  Fluid Button
</Button>
```

### Active

A button can be styled to indacte it is active.

```js live
<>
  <Button buttonType="primary">Normal</Button>{' '}
  <Button buttonType="primary" active>
    Active
  </Button>
</>
```
