## Examples

### Basic

A basic usage of Card wit an image.

```js live
// import skyline from '@assets/skyline.jpg';
// import bubbles from '@assets/bubbles.jpg';
// import trees from '@assets/trees.jpg';

<Container>
  <Row gutter={1}>
    <Col span={8}>
      <Card>
        <CardImage src={skyline} />
        <H3>Title</H3>
        <P>Card content with a short text containing a description.</P>
        <P>
          <Button buttonType="primary">Call To Action</Button>
        </P>
      </Card>
    </Col>
    <Col span={8}>
      <Card>
        <CardImage src={bubbles} />
        <H3>Title</H3>
        <P>Card content with a short text containing a description.</P>
        <P>
          <Button buttonType="primary">Call To Action</Button>
        </P>
      </Card>
    </Col>
    <Col span={8}>
      <Card>
        <CardImage src={trees} />
        <H3>Title</H3>
        <P>Card content with a short text containing a description.</P>
        <P>
          <Button buttonType="primary">Call To Action</Button>
        </P>
      </Card>
    </Col>
  </Row>
</Container>
```

### Interactive

When card is interactive it responds to mouse over and tab focus.

```js live
// import skyline from '@assets/skyline.jpg';
// import bubbles from '@assets/bubbles.jpg';
// import trees from '@assets/trees.jpg';

<Container>
  <Row gutter={1}>
    <Col span={8}>
      <Card interactive onClick={() => console.log('clicked skyline')}>
        <CardImage src={skyline} />
        <H3>Title</H3>
        <P>Card content with a short text containing a description.</P>
        <P>
          <Button buttonType="primary">Call To Action</Button>
        </P>
      </Card>
    </Col>
    <Col span={8}>
      <Card interactive onClick={() => console.log('clicked bubbles')}>
        <CardImage src={bubbles} />
        <H3>Title</H3>
        <P>Card content with a short text containing a description.</P>
        <P>
          <Button buttonType="primary">Call To Action</Button>
        </P>

    <Col span={8}>
      <Card interactive onClick={() => console.log('clicked trees')}>
```

### Overlay

An overlay appears on a mouse over.

```js live
// import skyline from '@assets/skyline.jpg';
// import bubbles from '@assets/bubbles.jpg';
// import trees from '@assets/trees.jpg';

  <Row gutter={1}>
    <Col span={8}>
      <Card
        overlay={
          <Text color="white">
            <h2>Skyline</h2>
          </Text>
        }
      >
        <CardImage src={skyline} />
      </Card>
    </Col>
    <Col span={8}>
      <Card
        overlay={

            <h2>Bubbles</h2>
          </Text>

      </Card>
    </Col>
    <Col span={8}>

```

## Design Guidelines

Use cards to group content in the ui.
