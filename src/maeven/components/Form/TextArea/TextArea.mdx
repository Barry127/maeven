## Examples

### Basic

A basic TextArea.

```js live
<TextArea placeholder="Type here...">
  Childeren are placed under the textarea
</TextArea>
```

### Autosize

Textarea autosizing can be on or off.

```js live
<Row gutter={1}>
  <Col transparent>
    <TextArea placeholder="I will grow in height as my input grows" />
  </Col>
  <Col transparent>
    <TextArea autoSize={false} placeholder="I will not grow in height" />
  </Col>
</Row>
```

### Disabled

Textareas can be disabled.

```js live
<Row gutter={1}>
  <Col transparent>
    <TextArea disabled value="Disabled" />
  </Col>
  <Col transparent>
    <TextArea readOnly value="Read Only" />
  </Col>
</Row>
```

### Size

Textareas can have different sizes.

```js live
<Row gutter={1}>
  <Col transparent span={8}>
    <TextArea size="sm" placeholder="sm" />
  </Col>
  <Col transparent span={8}>
    <TextArea size="md" placeholder="md" />
  </Col>
  <Col transparent span={8}>
    <TextArea size="lg" placeholder="lg" />
  </Col>
</Row>
```

### Error

A textarea can indicate it has an error.

```js live
<Row gutter={1}>
  <Col transparent span={12}>
    <TextArea hasError placeholder="Error..." />
  </Col>
  <Col transparent span={12}>
    <TextArea hasError placeholder="Error...">
      Text indicating error
    </TextArea>
  </Col>
</Row>
```

## Design Guidelines
