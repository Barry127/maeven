# ThemeProvider

ThemeProvider creates a React context Provider for a Maeven Theme.

## Props

<Props of="ThemeProvider" />

## Examples

### Basic

```js live
<ThemeProvider theme={MaevenDefault}>
  <Block>
    All child components and deeply nested components have this theme.
  </Block>
</ThemeProvider>
```
