# CSS Variables

Maeven will set most theme options als css variables so consistent styling is easy to extend in your own custom classes. Below a list of all Maeven CSS variables.

## Base

| Name         | Description                                                                 |
| :----------- | :-------------------------------------------------------------------------- |
| `--mvn-base` | Base size for all theme. This is the value `1rem` will be. defaults to 10px |

## Animations

| Name                               | Description                             |
| :--------------------------------- | :-------------------------------------- |
| `--mvn-transition-duration`        | Default duration for transitions        |
| `--mvn-transition-timing-function` | Default timing function for transitions |
