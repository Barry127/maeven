# Grid

Maeven comes with a full responsive 24 column grid support.

## Examples

### Basic

A basic grid

```js live
<Container fluid>
  <Row>
    <Col span={24} background="indigo">
      Col-24
    </Col>
  </Row>
  <Row>

      Col-12
    </Col>
    <Col background="darkGrey" span={12}>
      Col-12
    </Col>
  </Row>
  <Row>
    <Col background="indigo" span={8}>
      Col-8
    </Col>
    <Col background="darkGrey" span={8}>
      Col-8
    </Col>
    <Col background="indigo" span={8}>
      Col-8
    </Col>
  </Row>
  <Row>
    <Col background="indigo" span={6}>
      Col-6
    </Col>
    <Col background="darkGrey" span={6}>
      Col-6
    </Col>
    <Col background="indigo" span={6}>
      Col-6

    <Col background="darkGrey" span={6}>
      Col-6
```

### Alignment

Columns can be vertically aligned.

```js live
<Container fluid>
  <Heading level="h4" size="h5">
    Top (default)
  </Heading>
  <Row align="top">
    <Col background="indigo" span={8} style={{ height: 75 }}>
      Col-8
    </Col>
    <Col background="darkGrey" span={8} style={{ height: 50 }}>
      Col-8
    </Col>
    <Col background="indigo" span={8} style={{ height: 150 }}>
      Col-8
    </Col>
  </Row>
  <Heading level="h4" size="h5">
    center
  </Heading>
  <Row align="center">
    <Col background="indigo" span={8} style={{ height: 75 }}>
      Col-8
    </Col>
    <Col background="darkGrey" span={8} style={{ height: 50 }}>
      Col-8
    </Col>
    <Col background="indigo" span={8} style={{ height: 150 }}>
      Col-8
    </Col>
  </Row>
  <Heading level="h4" size="h5">
    Bottom
  </Heading>
  <Row align="bottom">
    <Col background="indigo" span={8} style={{ height: 75 }}>
      Col-8
    </Col>
    <Col background="darkGrey" span={8} style={{ height: 50 }}>
      Col-8
    </Col>
    <Col background="indigo" span={8} style={{ height: 150 }}>
      Col-8
    </Col>
  </Row>

```

### Resizing

Grid Columns can resize in different viewports

```js live
<Container fluid>
  <Row>
    <Col background="indigo" span={2} xs={24} sm={8} lg={6}>
      Col 2 / xs-24 / sm-8 / lg-6
    </Col>
    <Col background="darkGrey" span={2} xs={12} sm={8} lg={6}>
      Col 2 / xs-12 / sm-8 / lg-6
    </Col>
    <Col background="indigo" span={2} xs={12} sm={8} lg={12}>
      Col 2 / xs-12 / sm-8 / lg-12
    </Col>
  </Row>
</Container>
```

### Wrapping

Columns wrap to to a new line. But this can be prevented.

```js live
<Container fluid>
  <Heading level="h4" size="h5">
    Wrap (default)
  </Heading>
  <Row gutter={2}>
    <Col background="indigo" span={6}>
      Col-6
    </Col>
    <Col background="darkGrey" span={6}>
      Col-6
    </Col>
    <Col background="indigo" span={6}>
      Col-6
    </Col>
    <Col background="darkGrey" span={8}>
      Col-8
    </Col>
  </Row>
  <Heading level="h4" size="h5">
    No wrap
  </Heading>
  <Row style={{ overflowX: 'scroll' }} gutter={2} wrap={false}>
    <Col background="indigo" span={6}>
      Col-6
    </Col>
    <Col background="darkGrey" span={6}>
      Col-6
    </Col>
    <Col background="indigo" span={6}>
      Col-6
    </Col>
    <Col background="darkGrey" span={8}>
      Col-8
    </Col>
  </Row>
</Container>
```

### Adaptive

Columns without a span resize adaptive to fill up a row.

```js live
<Container fluid>
  <Row>
    <Col background="indigo" span={4}>
      Col-4
    </Col>
    <Col background="darkGrey">Col</Col>
  </Row>
  <Row>
    <Col background="indigo">Col</Col>
    <Col background="darkGrey" span={12}>
      Col-12
    </Col>
    <Col background="indigo">Col</Col>
  </Row>
</Container>
```

### Hidden

Columns can be (selectively) hidden.

```js live
<Container fluid>
  <Row>
    <Col background="indigo" span={8} hidden>
      Col-8 hidden
    </Col>
    <Col background="darkGrey" span={16}>
      Col-16
    </Col>
  </Row>
  <Row>
    <Col background="indigo" span={8}>
      Col-8
    </Col>
    <Col background="darkGrey" span={16} hidden="md">
      Col-16 hidden from md and down
    </Col>
  </Row>
  <Row>
    <Col background="indigo" span={8}>
      Col-8
    </Col>
    <Col background="darkGrey" span={16} hidden={['sm', 'lg']}>
      Col-16 hidden on sm and lg
    </Col>
  </Row>
</Container>
```
