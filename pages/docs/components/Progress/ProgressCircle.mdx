# ProgressCircle

<Description of="ProgressCircle" />

## Props

<Props of="ProgressCircle" />

## Examples

### Basic

A basic usage of ProgressCircle.

```js live
<ProgressCircle value={50}>
  <H3>50%</H3>
</ProgressCircle>
```

### Size

ProgressCircles can have different sizes.

```js live
<Row align="center">
  <Col style={{ textAlign: 'center' }}>
    <ProgressCircle size={92} value={50}>
      <H3>50</H3>
    </ProgressCircle>
  </Col>
  <Col style={{ textAlign: 'center' }}>
    <ProgressCircle size={128} value={50}>
      <H2>50</H2>
    </ProgressCircle>
  </Col>
  <Col style={{ textAlign: 'center' }}>
    <ProgressCircle size={160} value={50}>
      <H1>50</H1>
    </ProgressCircle>
  </Col>
</Row>
```

### Indeterminate

A ProgressCircle can indicate an Indeterminate state.

```js live
<ProgressCircle indeterminate>Loading data</ProgressCircle>
```
