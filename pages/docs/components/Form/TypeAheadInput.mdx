## Examples

### Basic

A basic usage of TypeAheadInput.

```js live
<TypeAheadInput
  items={['JavaScript', 'Java', 'PHP', 'Perl', 'C', 'C++', 'Rust', 'Lua']}
>
  Start typing...
</TypeAheadInput>
```

### Controlled

A controlled TypeAheadInput.

```js live withRender
const Controlled = (props) => {
  const [value, setValue] = useState('');
  return (
    <TypeAheadInput
      {...props}
      value={value}
      onChange={(ev) => setValue(ev.target.value)}
    >
      Value: <i>{value}</i>
    </TypeAheadInput>
  );
};

render(
  <Row gutter={1}>
    <Col>
      <H4>Controlled input with items as array</H4>
      <Controlled
        items={['JavaScript', 'Java', 'PHP', 'Perl', 'C', 'C++', 'Rust', 'Lua']}
      />
    </Col>
    <Col>
      <H4>Controlled input with items as syncGetItems</H4>
      <Controlled
        items={() => [
          'JavaScript',
          'Java',
          'PHP',
          'Perl',
          'C',
          'C++',
          'Rust',
          'Lua'
        ]}
      />
    </Col>
  </Row>
);
```

### Async items

Items can be loaded async.

```js live withRender
const items = ['JavaScript', 'Java', 'PHP', 'Perl', 'C', 'C++', 'Rust', 'Lua'];
const getItems = () =>
  new Promise((resolve) => {
    setTimeout(() => {
      resolve(items);
    }, 500);
  });

render(<TypeAheadInput items={getItems} />);
```
